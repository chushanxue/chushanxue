### 1、数组去重

Set 是 JavaScript 中一个内建的对象，它只允许存储唯一的值。利用这一特性，我们可以轻松实现数组去重。

```js
const unique = (arr) => {
  return [...new Set(arr)];
};
```

使用 filter，遍历数组，并对每个元素执行一个测试函数，只有使测试函数返回 true 的元素才会被保留在新数组中。

```js
const unique = (arr) => {
  return arr.filter((item, index) => arr.indexOf(item) === index);
};
```
