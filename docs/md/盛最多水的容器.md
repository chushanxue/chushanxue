### 一、题目

给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

**找出其中的两条线**，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

![ ](/md/盛最多水的容器/1.jpg)

```bash
示例 1：
输入：[1,8,6,2,5,4,8,3,7]
输出：49
解释：图中垂直线代表输入数组 [1,8,6,2,5,4,8,3,7]。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

示例 2：
输入：height = [1,1]
输出：1
```

### 二、分析

注意这个题和接雨水有着本质的区别，接雨水需要算出所有的雨水，而这道题只需要找出其中两条可以容纳最多的水的线，算出容器可以储存的最大水量。

还是用双指针的思想，这次需要移动短边，因为移动长边的话，无论如何，容积都不会增加，因为短边才是**限制**，移动短边才有可能增加容积

### 三、正确答案

```js
/**
 * @param {number[]} height
 * @return {number}
 */
var maxArea = function (height) {
  const n = height.length;
  let left = 0;
  let right = n - 1;
  let ans = 0; //最大存水量
  let area; //当前存水量
  while (left < right) {
    // 容器装水的面积
    area = (right - left) * Math.min(height[left], height[right]);
    ans = Math.max(area, ans); //重点在这句，此题无需结束遍历，全部遍历结束才能得到最终的ans
    if (height[left] < height[right]) {
      left++;
    } else {
      right--;
    }
  }

  return ans;
};
```
