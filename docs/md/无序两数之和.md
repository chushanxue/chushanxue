### 一、题目

给定一个整数数组 nums 和一个整数目标值 target，请你在该数组中找出 和为目标值 target 的那 两个 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案，并且你不能使用两次相同的元素。

你可以按任意顺序返回答案。

示例 ：

```bash
输入：nums = [2,7,11,15], target = 9
输出：[0,1]
解释：因为 nums[0] + nums[1] == 9 ，返回 [0, 1] 。

输入：nums = [3,2,4], target = 6
输出：[1,2]

输入：nums = [3,3], target = 6
输出：[0,1]
```

### 二、解题思路

注意题目中的**按任意顺序**，一般来说，顺序不重要，就需要我们规定一个顺序。

利用双指针，分别从两端开始向中间移动，如果和大于目标值，就把最大的去掉，则右指针向左移动，如果和小于目标值，就把最小的去掉，则左指针向右移动，直到找到和等于目标值的两个数，返回它们的下标。

这种思路的前提是有序数组，但如果我们对数组直接`sort`排序，那么原数组就会被改变，而最终我们要返回的还是原数组的下标。

所以我们可以利用对象数组，一开始就把原数组的值`value`和对应的下标`index`存储成为对象`{value:value,index:index}`，然后再对对象数组进行排序，此时原来的index已经变成元素对象里的值，不会被排序所影响。

需要注意的点就是不要混淆`index`和`left`，`right`，因为此时`left`和`right`才是下标，而`index`已经是元素对象里的值

### 三、正确答案

```js
var twoSum = (nums, target) => {
  let left = 0;
  let right = nums.length - 1;

  let ArraySums = nums
    .map((value, index) => ({ value, index }))
    .sort((a, b) => a.value - b.value);

  // 如果前后指针还没有相遇的话
  while (left < right) {
    let sum = ArraySums[left].value + ArraySums[right].value;
    if (sum === target) break;
    if (sum > target) right--;
    if (sum < target) left++;
  }

  return [ArraySums[left].index, ArraySums[right].index];
};
```
